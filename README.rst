===============================
pdftables
===============================


Extract tables from pdf documents.
`pdftables` is a fork of the pdftables developed by `ScraperWiki
<http://scraperwiki.com>`_

* Free software: BSD license
* Documentation: http://pdftables.readthedocs.org.
